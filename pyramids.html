Pyramids

I created all assets for this scene in Maya and rendered the images using Arnold. I modelled the pillars as separate parts so that I could use them both as intact pillars and as collapsed parts. To create the tents, I created simple frames onto which I placed plane meshes and used nConstraints to attach the edges. Using nCloth simulations, I simulated the fabric of the tents moving in the wind. I used basic material shaders for the pyramids/tents and free textures for the pillars and sand.
