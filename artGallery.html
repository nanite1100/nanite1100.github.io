Art gallery

I modelled and textured all assets for this scene in Maya. I created the details for the frames by using the quad draw tool and live surface so that the details would conform to the shape of the frame mesh. Because galleries don't usually display empty frames, I sourced digitals scans of paintings which are within the public domain and used them to texture mesh planes arranged within each of the frames. Lastly, I used Maya's ncloth system to create physically realistic fabric banners and curtains. I plan to add more assets to improve the scene including decorations on the tables and light fixtures.
