Jungle temple

I created this scene in Unity using free assets from the Unity store. I used Unity's terrain tools to create, shape, and texture the ground plane. I then populated the terrain surface with palm trees, shrubs, and different flower textures using the terrain tools. I created the main area of the scene by placing the building assets as well as other decorative assets to enhance the scene (rocks, pillars, candles, clay pottery). I've been learning more about how post processing can be implemented and so I added various effects to the camera including a bloom effect, colour grading, and depth of field (really makes it all look so much better).
