Modern seating area

This scene was modelled and shaded in Maya. I modelled all objects in the scene and also used a boolean function to create holes in the walls for the windows and skylight. I used Maya's Generate system to create flowers and placed them within the vases. Lastly, I assigned basic shaders for all the assets in the scene. I rendered this image using the Arnold renderer.
